# 版本路线图

## 当前版本结论

- 当前版本：`V0.5-beta`
- 当前定位：`作品集主站 + 知识总结栏目`
- 当前状态：主站信息架构、真实项目接入和一轮招聘视角 UI/UX 收口已完成

## 版本演进

### V0.1-alpha
- 项目初始化
- 文档骨架建立
- 目录结构和运行骨架准备

### V0.2-alpha
- 产品定义
- 信息架构、页面清单、字段定义、内容模型补齐

### V0.3-beta
- 前台页面与内容闭环
- Notes / Downloads / Topics 最小闭环打通
- 本地 Markdown 导入与 draft 管理能力建立

### V0.4-beta
- 测试验收文档收口
- 本地验收清单形成

### V0.5-beta
- 站点定位从知识站升级为作品集主站 + Notes 栏目
- 新增 `Projects` 列表页与项目详情页模板
- 首页升级为 portfolio 首页
- About / Resume 改为求职展示视角页面
- 真实项目内容、截图证据和项目外链接入
- Projects 相关页面完成一轮招聘视角 UI/UX 收口

## 当前版本包含

- 主导航：`Projects / Notes / Resume / GitHub / About`
- 二级入口：`Topics / Downloads` 保留但降级
- 项目详情页：总览、截图证据、边界说明、面试追问
- 首页：代表作优先、知识内容保留、外链层级明确
- 本地维护：Markdown 导入与 draft 管理继续可用
