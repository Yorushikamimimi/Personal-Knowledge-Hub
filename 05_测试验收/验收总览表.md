# 验收总览表

## 文档说明

本文档用于阶段 4 的整体验收收口，覆盖当前 `V0.4-beta` 已完成的前台页面、内容闭环、首页联调和基础布局能力。

适用范围：
- 首页精选区
- Notes 列表页 / 详情页
- Downloads 列表页 / 详情页
- Topics 聚合页
- About / Privacy / Copyright / Disclaimer
- layout / Header / Footer / PageContainer

使用方式：
- 执行验收前先完成本地安装与构建。
- 每项验收完成后填写结果、日期和备注。
- 如发现问题，在备注中记录复现路径和影响范围。

## 页面级验收清单

### 1. 首页精选区

- [ ] 首页可正常访问。
- [ ] Hero 区文案显示正常。
- [ ] Notes 精选区显示最新 published 内容。
- [ ] Downloads 精选区显示最新 published 内容。
- [ ] Topics 精选区显示 published Topic。
- [ ] 精选区内容点击后可跳转到对应页面。
- [ ] “查看全部”类入口可跳转到列表页。

### 2. Notes 模块

- [ ] `/notes` 列表页只显示 `status=published` 的文章。
- [ ] 列表项显示 `title / summary / category / tags / publishedAt 或 updatedAt`。
- [ ] 列表默认按时间倒序。
- [ ] `/notes/[slug]` 可正常渲染正文。
- [ ] 详情页顶部存在“返回笔记列表”按钮。
- [ ] 点击返回按钮可跳转回 `/notes`。
- [ ] draft note 直接访问时进入 notFound 场景。

### 3. Downloads 模块

- [ ] `/downloads` 列表页只显示 `status=published` 的资源。
- [ ] 列表项显示 `title / summary / fileType / version / size / updatedAt / tags`。
- [ ] `/downloads/[slug]` 详情页字段完整。
- [ ] 下载按钮可访问真实文件。
- [ ] draft download 不出现在列表，也不进入静态详情页。

### 4. Topics 模块

- [ ] `/topics` 可正常访问。
- [ ] 只显示 `status=published` 的 Topic。
- [ ] 每个 Topic 显示 `title / summary / order`。
- [ ] 每个 Topic 可展示关联 Note / Download 标题与链接。
- [ ] draft topic 不显示。

### 5. 合规与说明页面

- [ ] `/about` 页面结构完整，可用于对外展示。
- [ ] `/privacy` 页面结构完整。
- [ ] `/copyright` 页面结构完整。
- [ ] `/disclaimer` 页面结构完整。

### 6. 全局布局

- [ ] Header 导航完整，包含首页、知识笔记、专题、下载资源、关于。
- [ ] Footer 包含 About、Privacy、Copyright、Disclaimer、ICP 展示位。
- [ ] PageContainer 宽度与留白统一。
- [ ] 各页面排版风格一致。
