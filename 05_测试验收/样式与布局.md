# 样式与布局

## 文档说明

本文档用于验证当前站点的基础展示一致性，不聚焦视觉优化方案，而是聚焦可读性、结构稳定性和现有布局是否符合博客 / 文档风格。

## 验证范围

- layout.tsx 挂载的全局结构

- 首页精选区布局
- Notes / Downloads / Topics 卡片展示
