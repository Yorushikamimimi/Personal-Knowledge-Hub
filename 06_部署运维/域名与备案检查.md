# 域名与备案检查

## 1. 当前状态（2026-03-21）

- ICP 主体状态：`正常`
- 网站接入状态：`正常`

- 公安网备：已生成数据码

说明：从备案侧已满足上线前置条件。

## 2. 发布前检查清单

### 2.1 DNS 解析

```bash
dig +short yoruming.cn
dig +short www.yoruming.cn
```

预期：返回你的云服务器公网 IP。

### 2.2 HTTP 可达性

```bash
curl -I http://yoruming.cn
curl -I http://www.yoruming.cn
```

预期：返回 `200` 或 `301/302`。

### 2.3 备案号展示

网站页脚需展示：

- ICP 备案号（例如：蜀ICP备2026007917号）
- 公安网备信息（按规范展示）

## 3. 风险点

- 域名虽备案正常，但若 DNS 未指向当前服务器，会导致外网不可达
- 若 Nginx 里 `server_name` 重复，会出现 `conflicting server name`
- 若只配了 `www` 未配裸域，会出现一部分用户访问失败
