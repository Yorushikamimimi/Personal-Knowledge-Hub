# 信息架构 IA

## 顶层结构

```text
首页 /
├─ Projects /projects
│  └─ Project Detail /projects/[slug]
├─ Notes /notes
│  └─ Note Detail /notes/[slug]
├─ Resume /resume
├─ GitHub (外链)
├─ About /about
├─ Topics /topics
├─ Downloads /downloads
│  └─ Download Detail /downloads/[slug]
└─ 合规页
   ├─ /privacy
   ├─ /copyright
   └─ /disclaimer
```

## 信息层级说明

### 第一层：求职主路径

- 首页
- Projects
- Resume
- GitHub
- About

这一层负责快速建立个人定位和项目能力认知。

### 第二层：知识沉淀路径

- Notes
- Topics
- Downloads

这一层负责承接长期内容、知识结构和配套资料。

## 核心访问路径

### 路径 1：招聘方快速浏览

首页 -> Projects -> Project Detail -> GitHub / Resume

### 路径 2：查看技术积累

首页 -> Notes -> Note Detail

### 路径 3：查看知识地图与配套资料

首页 -> Topics / Downloads
