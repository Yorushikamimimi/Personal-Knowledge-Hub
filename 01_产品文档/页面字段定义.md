# 页面字段定义

## 首页 `/`

### Hero

- `headline`：一句话定位
- `roleSummary`：后端 / 全栈方向说明
- `techKeywords[]`：技术关键词
- `primaryLinks[]`：Projects / Notes / GitHub / Resume

### Featured Projects

- `title`
- `summary`
- `role`
- `techStack[]`
- `highlights[]`
- `detailUrl`
- `githubUrl`

### Latest Notes

- `title`
- `summary`
- `category`
- `tags[]`
- `updatedAt`
- `detailUrl`

## Projects `/projects`

- `title`
- `slug`
- `summary`
- `role`
- `techStack[]`
- `highlights[]`
- `githubUrl`
- `detailUrl`
- `featured`
- `sortOrder`

## Project Detail `/projects/[slug]`

- `title`
- `summary`
- `role`
- `goals[]`
- `responsibilities[]`
- `highlights[]`
- `techStack[]`
- `screenshotNote`
- `boundaries[]`
- `githubUrl`
- `readmeUrl`
- `demoUrl`

## About `/about`

- `intro`
- `jobDirection`
- `techStack[]`
- `engineeringFocus[]`
- `externalLinks[]`

## Resume `/resume`

- `resumeStatus`
- `resumeLink`
- `fallbackLinks[]`
