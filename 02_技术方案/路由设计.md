# 路由设计

## 当前路由总览

| 路由 | 页面类型 | 内容来源 | 静态参数 |
| --- | --- | --- | --- |
| `/` | 静态页面 | 站点固定文案 + Projects / Notes 聚合 | 否 |
| `/projects` | 静态页面 | `content/projects` | 否 |
| `/projects/[slug]` | 动态分段静态生成 | `content/projects/*.mdx` | 是 |
| `/notes` | 静态页面 | `content/notes` | 否 |
| `/notes/[slug]` | 动态分段静态生成 | `content/notes/*.mdx` | 是 |
| `/topics` | 静态页面 | `content/topics` | 否 |
| `/downloads` | 静态页面 | `content/downloads` | 否 |
| `/downloads/[slug]` | 动态分段静态生成 | `content/downloads/*.mdx` | 是 |
| `/about` | 静态页面 | 固定文案 | 否 |
| `/resume` | 静态页面 | 固定文案 / 占位信息 | 否 |
| `/privacy` | 静态页面 | 固定文案 | 否 |
| `/copyright` | 静态页面 | 固定文案 | 否 |
| `/disclaimer` | 静态页面 | 固定文案 | 否 |

## 本地维护路由

| 路由 | 用途 | 环境 |
| --- | --- | --- |
| `/import-notes` | Markdown 导入页 | 仅开发环境 |
| `/draft-notes` | draft 列表页 | 仅开发环境 |
| `/draft-notes/[slug]` | draft 详情页 | 仅开发环境 |
| `/api/import-notes/save` | 保存导入结果 | 仅开发环境 |

## 主导航规则

- 主导航只保留 `Projects / Notes / Resume / GitHub / About`
- `Topics / Downloads` 下沉为首页和页脚中的二级入口

## 静态参数生成规则

- `/projects/[slug]`：扫描所有 `status=published` 的 Project
- `/notes/[slug]`：扫描所有 `status=published` 的 Note
- `/downloads/[slug]`：扫描所有 `status=published` 的 DownloadResource
